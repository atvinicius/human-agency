# Human Agency

**The Orchestration Layer for the AI Age**

One Human. One Million Agents.

## About

Human Agency is building the interface that transforms humans from bottlenecks into Sovereign Operators. We're bridging the gap from 1:1 (one human, one tool) to 1:1M (one human, one million agents).

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy

Deploy to Vercel:
```bash
npx vercel
```
